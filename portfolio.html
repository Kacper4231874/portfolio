Projekty INF.02

Realizacja zadań kwalifikacyjnych

07
  • 2025

    Konfiguracja sieci LAN i Administracja systemami operacyjnymi.

    Projektowanie i wdrażanie infrastruktury sieciowej: Tworzenie adresacji IP, definiowanie podsieci oraz konfiguracja VLAN-ów w celu segmentacji ruchu sieciowego. Konfiguracja urządzeń sieciowych: Praktyczna obsługa przełączników (switchy) i routerów (głównie w środowisku Cisco), w tym konfiguracja interfejsów, routing, listy ACL (Access Control Lists) oraz podstawowych zabezpieczeń firewall. Kontrola dostępu: Implementacja mechanizmów uwierzytelniania i autoryzacji użytkowników oraz urządzeń w sieci. Instalacja i administracja serwerami: Instalacja, konfiguracja podstawowa i zarządzanie systemami operacyjnymi serwerowymi, takimi jak Windows Server oraz Ubuntu Server. Zarządzanie usługami sieciowymi: Konfiguracja, optymalizacja i utrzymanie kluczowych usług, w tym: DHCP: Automatyczne przydzielanie adresów IP klientom. DNS: Rozwiązywanie nazw domenowych na adresy IP i odwrotnie. FTP: Udostępnianie plików w sieci. WWW: Hostowanie stron i aplikacji internetowych. Zarządzanie tożsamością: Tworzenie kont użytkowników i grup, przypisywanie uprawnień oraz zarządzanie zasobami w środowiskach takich jak Active Directory (Windows) czy OpenLDAP/Kerberos (Linux). Kluczowe technologie: Cisco Packet Tracer / IOS, Active Directory, Ubuntu Server (w tym usługi jak Bind9, ISC DHCP, Apache2/Nginx, vsftpd), Zapory sieciowe (Firewall - np. Windows Firewall, UFW, iptables/nftables).

    Cisco Active Directory Ubuntu Server Firewall
  • 2025

    Wirtualizacja i projekty złożone

    Konfiguracja środowisk wirtualnych: Instalacja, konfiguracja i zarządzanie maszynami wirtualnymi (VM) oraz sieciami wirtualnymi łączącymi te maszyny. Pozwala to na symulację złożonych topologii bez konieczności posiadania fizycznego sprzętu. Realizacja kompleksowych projektów: Wykorzystanie wirtualizacji do budowy i testowania całych środowisk, takich jak: Sieć firmowa z VLAN-ami: Projektowanie i wdrażanie segmentowanej sieci przedsiębiorstwa z różnymi wydziałami (VLANy) oraz odpowiednimi regułami routingu i bezpieczeństwa między nimi. Serwer plików z zaawansowaną kontrolą dostępu: Konfiguracja serwera plików (np. Windows File Server lub Samba na Linux) z precyzyjnym przypisywaniem uprawnień dla użytkowników i grup na poziomie folderów i plików. Strategie backupu danych: Projektowanie i implementacja rozwiązań do regularnego tworzenia i weryfikacji kopii zapasowych krytycznych danych z serwerów. Kluczowe technologie: Platformy wirtualizacyjne - VirtualBox, VMware (np. Workstation Player/Pro, ESXi), Symulacja sieci (np. Cisco Packet Tracer - SNUP).

    VirtualBox SNMP VMware
